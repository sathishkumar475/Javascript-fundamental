JavaScript everywhere

1995 ─ Brendan Eich creates JavaScript (Mocha → LiveScript → JavaScript)
1996 ─ Microsoft releases JScript (browser wars start)
1997 ─ ECMAScript 1 (ES1) – first standard
1999 ─ ECMAScript 3 (ES3) – stable & widely used
2000-2004 ─ ES4 abandoned → "Dark Age" of JavaScript
2005 ─ AJAX popularized (Gmail, Google Maps), jQuery arrives
2009 ─ ECMAScript 5 (ES5), Node.js released → server-side JS
2015 ─ ECMAScript 6 (ES6) – huge update (classes, promises, let/const, modules)
2016+ ─ Annual updates (ES7, ES8, …)
- 2017 (ES8): async/await
- 2020 (ES11): optional chaining, nullish coalescing
- 2022+: top-level await, pattern matching proposals
Today ─ JavaScript runs everywhere (web, server, mobile, desktop, IoT, AI)
