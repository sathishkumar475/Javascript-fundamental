What is meant by data type?.

A data type is an attribute associated with a piece of data that tells a computer system how to interpret its value. Understanding data types ensures that data is collected in the preferred format and the value of each property is as expected.

Data type

let num = 42; // Number
let str = "JavaScript"; // String
let isCool = true; // Boolean
let notAssigned; // Undefined
let emptyValue = null; // Null
let uniqueKey = Symbol(); // Symbol
let largeNum = 9007199254740991n; // BigInt

  • Number: For any numeric value (e.g., 5, 3.14, -42).
  • String: For text (e.g., "Hello", 'World').
  • Boolean: For true/false values (e.g., true, false).
  • Undefined: When a variable is declared but not assigned a value (e.g., let x;).
  • Null: Represents an empty or non-existent value (e.g., let y = null;).
  • Symbol: A unique and immutable value often used as object keys (e.g., Symbol("id")).
  • BigInt: For very large integers beyond the limit of the Number type (e.g., 123n).
